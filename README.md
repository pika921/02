# Mini Block Drop（かわいいテトリス風ミニゲーム）

スマホでも遊べるように、画面下に**タッチ操作キー**を追加しています。さらに、BGM・操作音・アクション音つきです。

## 遊び方

### 1) GitHub Pages で遊ぶ（おすすめ）

- 公開URL: `https://<GitHubユーザー名>.github.io/<リポジトリ名>/`
- 初回はリポジトリの **Settings → Pages** で Source を **GitHub Actions** に設定

### 2) ローカルで遊ぶ

```bash
cd /workspace/02
python3 -m http.server 4173
```

ブラウザで <http://localhost:4173/> を開く。

## 操作方法

### キーボード
- ← → : 移動
- ↓ : 高速落下
- ↑ : 一気落下
- Space / Z : 回転
- P : 一時停止
- R : リスタート

### スマホ（画面上ボタン）
- ◀ / ▶ : 移動
- ▼ : 高速落下
- ⟳ : 回転
- ⤓ 一気落下 : ハードドロップ
- ⏯ : 一時停止

## サウンド
- BGM（ループ）
- 操作音（移動・回転）
- アクション音（ライン消去・一気落下・ゲームオーバー）
- 右側の `サウンドON/OFF` ボタンで切り替え
